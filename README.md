#  Digital Newspaper Application

A desktop-based **Digital Newspaper Application** developed using **Python**, **Tkinter**, and **Pillow**. The application provides a newspaper-style graphical interface that displays the latest India and International news along with visually designed news posters.

##  Project Overview

This project was built to practice **Python GUI development** using Tkinter. It presents news in a clean newspaper-themed layout with categorized sections, custom backgrounds, and news posters.

The application demonstrates how Python can be used to build attractive desktop applications without using web technologies.

##  Features

*  Newspaper-themed graphical user interface
* 🇮🇳 Top 10 India News section
*  Top 10 International News section
*  News posters displayed alongside headlines
*  Custom newspaper background
*  Displays the current edition/date
*  Organized layout with borders and headings
*  Built completely using Python Tkinter

##  Technologies Used

* Python 3.x
* Tkinter
* Pillow (PIL)

##  How to Run

1. Clone this repository

```bash
git clone https://github.com/yourusername/Digital-Newspaper.git
```

2. Navigate to the project folder

```bash
cd Digital-Newspaper
```

3. Install Pillow

```bash
pip install pillow
```

4. Run the application

```bash
 newspaper.py
```
